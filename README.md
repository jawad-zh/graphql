GraphQL Profile
A modern profile dashboard built with GraphQL that allows students to explore their school statistics, XP progression, audit ratios, project results, and achievements through interactive SVG graphs.

📖 About The Project
This project was created as part of the GraphQL module at Zone01 Oujda.
The objective was to build a personal profile page using the provided GraphQL API endpoint. The application authenticates users using JWT authentication and displays personalized school statistics with a clean and responsive UI.
The platform data is fetched from:


GraphQL API Endpoint


Authentication Endpoint

🔑 Authentication
The application uses Basic Authentication to obtain a JWT token from the authentication endpoint.
Supported login methods:


username:password


email:password


After login, all GraphQL requests are authenticated using:
Authorization: Bearer <JWT_TOKEN>
